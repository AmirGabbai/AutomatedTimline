Features List:
V - Automatically load events from events.json and derive categories from description keys
V - Dynamically size the timeline width, year labels, and event lanes based on the min/max years in the data
V - Prevent event overlap by distributing items across multiple vertical lanes
V - Build the categories menu on the fly with color-coded buttons for every detected category
V - Persist category visibility through the URL `hide` parameter and restore it on page load and browser navigation
V - Allow zooming while preserving the centered year so the view stays focused
V - Display a detailed modal on event click with years, colored category dots, and per-category descriptions
V - Embed YouTube videos directly inside the modal when events include YouTube links
V - List all non-YouTube resource links inside the modal footer
V - Provide previous/next controls plus click-outside, ESC, and close button actions for the modal
V - Show warning guidance when events.json cannot be fetched due to CORS or missing file issues
V - Detect events occurring in the same year and promote them into up to seven stacked lanes
V - Shift the base timeline downward automatically as extra lanes are added so overlapping rows stay visible
